    This project is a compilation of tools and algorithms for NLP using Crystal-lang.

    Installation

    1. Add the dependency to your shard.yml:
    dependencies:
  nlp:
    github: Yueyehua/nlp
    1. Run shards install

    Usage

    require "nlp"
