# nlp

This project is a compilation of tools and algorithms for NLP using
Crystal-lang.

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  nlp:
    github: Yueyehua/nlp
```
2. Run `shards install`

## Usage

```crystal
require "nlp"
```
